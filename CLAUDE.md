## Контактная форма → Telegram
- Принимает: имя, email, описание задачи (текст), опционально телефон.
- Отправка: серверная функция (Next.js API route или Edge), которая делает POST на `https://api.telegram.org/bot<TOKEN>/sendMessage` с `chat_id` владельца.
- Токен бота — **только** в env (`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`), никогда в клиенте.
- На клиенте: rate-limit + honeypot против ботов.

## Жёсткие правила
- **Никогда** не коммитить `.mcp.json` в git — он в `.gitignore`. Токены — только в env.
